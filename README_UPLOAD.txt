葉鹿下町自治会ホームページ GitHub Pages公開用フォルダ

このフォルダの中身をGitHubリポジトリへアップロードしてください。
フォルダそのものではなく、中に入っている index.html や styles.css などをアップロードします。

公開に必要な主なファイル:
- index.html
- news.html
- circular.html
- disaster.html
- schedule.html
- garbage.html
- styles.css
- app.js
- assets/
- documents/
- .nojekyll

GitHub Pages設定:
1. GitHubで新しいリポジトリを作る
2. このフォルダの中身をアップロードする
3. Settings > Pages を開く
4. Source を Deploy from a branch にする
5. Branch を main、フォルダを /root にする
6. Save を押す

公開URLの例:
https://ユーザー名.github.io/リポジトリ名/
